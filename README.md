# 简易版图书管理系统后台 java614

## 项目概述

本项目是一套简易版图书管理系统后台前端项目，基于Vue2框架，提供美观的界面及部分功能模块。其设计初衷是便于二次开发，可适用于各类后台管理系统的应用场景。

## 技术栈

- **前端框架：**
  - Vue2
  - Vue Router
  - Vuex
- **UI组件库：**
  - Element UI
- **数据模拟：**
  - MockJS
- **网络请求：**
  - Axios

### 核心亮点

- **Mock数据：** 内置MockJS模拟后端接口，便于前端独立运行及测试。
- **状态管理：** 使用Vuex进行状态管理，支持持久化存储。
- **路由权限：** 基于角色的动态路由权限控制。
- **主题切换：** 支持明暗主题切换功能。
- **组件封装：** 高度封装的通用组件，如分页、面包屑等。

## 功能模块

- **静态仪表盘：** 直观展示系统核心数据，支持多种数据可视化图表。
- **图书管理：**
  - 图书列表
  - 添加/编辑图书
  - 图书分类管理
- **会员管理：**
  - 会员列表
  - 添加/编辑会员
  - 会员详情查看

## 系统角色

- 系统管理员
- 图书管理员
- 会员用户

## 数据处理

- 内置MockJS模拟后端接口，前端可独立运行无需搭建后端。
- 使用Axios封装HTTP请求，支持请求/响应拦截。
- 数据持久化存储，登录状态保持。

## 二次开发

- 支持对接真实后端API，移除Mock数据。
- 可扩展更多业务功能，如借阅管理、预约系统、电子书阅读等。

## 运行环境

- Node.js 版本 >= 10.0.0
- npm 版本 >= 6.0.0

## 部署步骤

1. 克隆项目到本地
2. 安装依赖：`npm install`
3. 启动开发服务器：`npm run serve`
4. 打包构建：`npm run build`
5. 部署至服务器
